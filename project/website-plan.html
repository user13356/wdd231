Typograph:

1. Main Font: Gothic(sans-serif)

  • Usage: Gothic font going to be used for headings on site for a popping, fresh appearance for users to see diffrent parts of this site.
  • Added to:
    • Headings (h1, h2, h3, h4): All headings on the website will use Gothic for easy navigation.

2. Secondary Font: Source Sans (sans-serif)

  • Usage: Source Sans for body text and paragraphs, ensuring that the content is easy to read and visually appealing for users.
  • Applied to:
    • Body Text (p): Most paragraphs, information about Western Cape nature, easy to read text.
    • Links and Buttons: Clicking links and buttons will also be used. Source Sans to used in website design.
